# Amazon (AMZN) DCF Valuation Model

A comprehensive **Discounted Cash Flow (DCF) valuation model for Amazon (NASDAQ: AMZN)** built using FY2024 financial data to estimate intrinsic enterprise and equity value.

## Project Overview

This project evaluates Amazon’s valuation using a **5-year DCF model (2025–2029)** based on historical financial performance, projected free cash flows, and sensitivity analysis.

The model incorporates:

* Historical financial statement analysis (FY2020–FY2024)
* Revenue and EBITDA forecasting
* Free Cash Flow (FCF) projections
* Discounted Cash Flow (DCF) valuation
* WACC and Terminal Growth sensitivity analysis
* Segment-level performance evaluation
* Investment thesis and risk assessment

* **Gross Margin:** 48.8%

## Valuation Output

* **PV of Free Cash Flows:** $331B
* **PV of Terminal Value:** $1.45T
* **Enterprise Value:** $1.78T
* **Equity Value:** $1.80T
* **Implied DCF Price Target:** **$170/share**

## Sensitivity Analysis

A valuation sensitivity model was conducted using multiple **WACC (8.0%–10.5%)** and **Terminal Growth Rate (2.5%–4.0%)** scenarios to evaluate downside and upside valuation ranges.

## Investment Thesis

### Bull Case Drivers

* AWS growth fueled by AI/ML demand
* Expanding high-margin advertising business
* Margin expansion through automation and logistics optimization
* International profitability improvements
* Strong Prime ecosystem and network effects

### Key Risks

* Regulatory and antitrust scrutiny
* High capex intensity for AI infrastructure
* Increased cloud competition from Azure and Google Cloud
* Consumer spending slowdown
* Valuation premium risk

## Key Takeaways

The DCF model estimates Amazon’s intrinsic value at **$170/share**, supported by AWS profitability, advertising growth, margin expansion, and strong free cash flow generation. However, valuation remains sensitive to assumptions surrounding growth rates, discount rates, and long-term profitability.
